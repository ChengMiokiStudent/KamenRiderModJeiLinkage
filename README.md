# KamenriderModJeiLinkage 模组

## 安装说明

1. 安装 [Forge](https://files.minecraftforge.net/) 对应版本。
2. 下载模组文件。
3. 将模组文件放入 `.minecraft/mods` 文件夹。
4. 启动游戏并选择正确的配置文件。

## 使用方法

添加到装了联动项模组的游戏里
